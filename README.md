Cryptogram Clone
==============

Inspired by [this app](https://play.google.com/store/apps/details?id=com.jmsc.cryptogram)

## Run Locally

Clone the project

```bash
  git clone https://github.com/belsabbagh/cryptogram-clone.git
```

Go to the project directory

```bash
  cd cryptogram-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
